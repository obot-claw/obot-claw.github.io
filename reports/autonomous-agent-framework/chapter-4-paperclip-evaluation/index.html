Chapter 4: Paperclip evaluation

Recommendation: run a bounded self-hosted Paperclip pilot. Paperclip targets the layer we are missing: agent teams, tickets, heartbeats, budgets, approvals, audit trails, and bring-your-own agents.

Findings

DimensionFindingPilot implication
CostCore appears MIT/self-hostable; hosted product is paid.Start local self-hosted. Do not assume SaaS is free.
SecurityAdds a powerful agent control plane and secret surface.Local loopback, telemetry off, scoped tokens, one safe repo.
CompatibilityAdvertises OpenClaw and Codex support.Validate adapters hands-on before migration.
CompetitorsCodex/OpenHands/Claude are strong executors; GitHub Copilot is repo-native; LangGraph/AutoGen require custom build.Paperclip is the best candidate for orchestration, not necessarily coding execution.
